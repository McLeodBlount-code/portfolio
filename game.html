Fixed Shooter Retro Video Game

A 2D interactive game simmilar to the arcade game Galaga developed with C and C++ on an MBED microcontroller.

TLDR

What I learned:

  • C and C++ file manipulation
  • Complex breadboard using microcontrollers, ULCDs, 5 way tactile switches, and SC card readers
  • Doublely Linked List implementation and management
  • Game debugging

During the summer between my first and second year at tech, I developed a fixed shooter video game simmilar in ways to Galaga and Space Invaders to be displayed on a ULCD display connected via breadboard to an MBED microcontroller. The game had several key features to enhance gamplay. The "start" screen of the game allows you to change the color of the player ship and see the main game controls including learning how to move and fire missiles. In game features included the ability to pause the game, multiple levels of gameplay, and animations for when you defeat enemy ships (or when they defeat you).

Game development first began with creating the hardware it runs on. Pictured above is a blueprint of this system. Implementing this hardware came with several difficulties, mainly stemming from issues with breadboard size and hardware malfunctions.
